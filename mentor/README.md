# mentor

日本語専用のメンタリングエージェント。コードレビュー・Q&A 指導・学習ロードマップ・次ステップ提案を提供する。

## What It Does

`/mentor` を呼び出すと、ユーザーのレベル（初心者 / 中級者）を自動判定してスタイルを切り替えながらメンタリングを行う。

## Agent

| Agent | Description |
|-------|-------------|
| `mentor` | コードレビュー・Q&A 指導・学習ロードマップ・次ステップ提案（日本語専用） |

## Features

| 機能 | 概要 |
|------|------|
| コードレビュー | 問題点・改善策・良い点をフィードバック |
| Q&A 指導 | 概念・エラー・設計の質問に答えながら理解を促進 |
| 学習指導 | 目標・現在地に合ったロードマップを提示 |
| 次ステップ提案 | 「次に学ぶと良いこと」を1つ具体的に提示 |

## Modes

- **初心者モード（ソクラテス式）** — 答えを直接与えず、質問で気づきを促す
- **中級者モード（直接的）** — 問題点と根拠を明確に指摘し、トレードオフを示す
